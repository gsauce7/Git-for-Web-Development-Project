    2. What is HTML used for? HYPER TEXT MARKUP LANGUAGE IS USED FOR THE STRUCTURE OF WEB PAGES AND OTHER CONTENT. IT CONSISTS OF ELEMENTS LAID OUT LOGICALLY- SOME ARE NESTED AND SOME NOT, AND CONTENT IS PLACED INSIDE THE TAGS.
    3. What is an attribute and where do we put it? AN ATTRIBUTE PROVIDES ADDITIONAL INFORMATION ABOUT AN HTML ELEMENT AND IS ALWAYS IN THE OPENING TAG. EXAMPLE: HREF IS AN ATTRIBUTE THAT TELLS WHICH WEB PAGE THE ANCHOR TAG SHOULD BE LINKING TO.
    4. What is the h1 tag used for? How many times should I use it on a page? THE H1 TAG DEFINES THE MOST IMPORTANT HEADING AND USUALLY IS ONLY USED ONCE ON THE PAGE. IT IS BY DEFAULT THE LARGEST OF THE H TAGS.

    8. What elements can I add an id to? AN ID IS ONLY INTENDED TO BE USED ON ONE ELEMENT.
    9. How many times can I use the same id on a page? ONCE
    10. What is a class? A CLASS IS AN HTML ATTRIBUTE THAT MEANT FOR APPLYING TO MULTIPLE ELEMENTS. CSS CAN BE APPLIED TO MULTIPLE ELEMENTS AT ONCE THIS WAY.
    11. What elements can I add a class to? YOU CAN ADD THIS ATTRIBUTE TO ANY ELEMENT WHERE IT MAKES SENSE THAT YOU WOULD ALSO APPLY THIS CLASS OTHER ELEMENTS.
    12. How many times can I use the same class on a page? YOU CAN USE THIS AS MANY TIMES AS YOU WANT FOR ELEMENTS YOU WANT TO STYLE THE SAME.
